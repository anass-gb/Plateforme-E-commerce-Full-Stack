# 🛒 Plateforme E-commerce Full-Stack

## 📌 Description

Cette application est une plateforme e-commerce complète développée avec une architecture moderne **Full-Stack**.  
Elle permet aux utilisateurs de parcourir des produits, gérer un panier et passer des commandes via une interface web intuitive.

Le projet repose sur une séparation claire entre :
- **Backend** : API REST sécurisée
- **Frontend** : Interface utilisateur dynamique

---

## 🚀 Technologies utilisées

### 🔧 Backend
- Spring Boot
- Java
- API REST
- Spring Security (authentification & autorisation)

### 🎨 Frontend
- React
- JavaScript (ES6+)
- CSS moderne

### 🗄️ Base de données
- MySQL

### 🛠️ Outils
- Git (versioning)
- Postman (tests API)
- Maven

---

## ⚙️ Fonctionnalités

### 👤 Gestion des utilisateurs
- Inscription / Connexion
- Authentification sécurisée
- Gestion des rôles

### 🛍️ Catalogue produits
- Affichage des produits
- Détails des produits
- Recherche / filtrage (optionnel)

### 🛒 Panier
- Ajouter / supprimer des produits
- Mise à jour des quantités

### 📦 Commandes
- Validation de commande
- Suivi des commandes

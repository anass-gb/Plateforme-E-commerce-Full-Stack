# 🛒 Plateforme E-commerce Full-Stack

## 📌 Description

Cette application est une plateforme e-commerce complète développée avec une architecture moderne Full-Stack.
Elle permet aux utilisateurs de parcourir des produits, gérer un panier et passer des commandes via une interface web intuitive.

Le projet repose sur une séparation claire entre :

* Backend : API REST sécurisée
* Frontend : Interface utilisateur dynamique

---

## 🚀 Technologies utilisées

### 🔧 Backend

* Spring Boot
* Java
* API REST
* Spring Security

### 🎨 Frontend

* React
* JavaScript (ES6+)
* CSS moderne

### 🗄️ Base de données

* MySQL

### 🛠️ Outils

* Git
* Postman
* Maven

## ⚙️ Fonctionnalités

### 👤 Gestion des utilisateurs

* Inscription / Connexion
* Authentification sécurisée
* Gestion des rôles

### 🛍️ Catalogue produits

### 🛒 Panier

## ▶️ Installation & Exécution

### 1️⃣ Cloner le projet

git clone https://github.com/anass-gb/Plateforme-E-commerce-Full-Stack.git
cd ton-repo

### 2️⃣ Backend (Spring Boot)

cd backend
mvn spring-boot:run

### 3️⃣ Frontend (React)

cd frontend
npm install
npm start
